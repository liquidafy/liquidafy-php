# Code of Conduct

This project is a professional space. We keep it simple:

- Be respectful and assume good faith. Disagree with ideas, never attack people.
- No harassment, discrimination, personal attacks, or demeaning language — anywhere in issues, PRs, reviews, or discussions.
- Keep discussions technical and on-topic; critique code and arguments, not contributors.
- Respect maintainers' time: search before opening issues, provide reproducible reports, accept that final API decisions rest with the maintainers.
- Never publish other people's private information or secrets (yours included — rotate any leaked key).

Violations can be reported privately to **support@liquidafy.com**. Maintainers may edit, lock, or remove content and ban contributors who break these rules.
